What Are Cursor / Windsurf Rules?

Rules are persistent instructions written into your AI editor to guide code style, stack preferences, and project conventions. In Cursor, place them in .cursor/rules. In Windsurf, import them through the Rules panel. This site collects ready-made rule templates — click Copy Rule to use them.

Covered use cases

  • Next.js full-stack and React Native mobile
  • Python web scraping and AI/ML engineering
  • Supabase and Go API backend development
  • Tailwind CSS frontend and Rust systems programming

Tip: Trim sections after copying to fit your project and avoid overrunning the AI context window.
